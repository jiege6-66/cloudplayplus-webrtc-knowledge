## 核心技术

CloudPlayPlus 使用的核心 WebRTC 技术包括：

- 信令服务器（基于 WebSocket）
- STUN/TURN 服务器配置
- 多设备流管理
- 会话控制
- 基于 JWT 的身份验证
- 跨平台实现（基于 Flutter）
